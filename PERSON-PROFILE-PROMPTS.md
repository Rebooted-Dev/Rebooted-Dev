## How To Use

1. Pick a prompt based on what you need.
2. Replace placeholders in `[brackets]`.
3. Run the prompt in ChatGPT, Gemini, or Claude.
4. Edit the output to sound like your real voice.

---

## Prompt 1: Profile Headline

```text
You are a profile writing coach.

Write 10 profile headline options for me.
Tone: clear, credible, warm, not hype.
Length: 8-16 words each.

My details:
- Name: [your name]
- Core roles: [e.g., builder, pastor, educator]
- Domain: [e.g., AI tools, ministry tech, accessibility]
- Audience: [who you want to attract]
- Location: [city/country]

Return:
1) 10 options
2) top 3 recommended with 1-line rationale each
```

## Prompt 2: About Section (Short + Long)

```text
Act as an executive profile editor.

Write two "About Me" versions:
1) Short version (70-90 words)
2) Long version (170-220 words)

Requirements:
- Keep it human and story-driven.
- Avoid buzzwords and cliches.
- Highlight lived experience + current mission.
- End with one sentence inviting collaboration.

My inputs:
- Background timeline: [key milestones]
- Current focus: [what you are building now]
- Communities served: [who benefits from your work]
- Values: [2-4 values]
```

## Prompt 3: Project Summary Table Copy

```text
You are helping me write concise project blurbs for GitHub/LinkedIn.

For each project, produce:
- one-sentence summary (max 18 words)
- status label (Active, Building, Research, Live)
- one suggested CTA line (e.g., "Try it", "View repo", "Join waitlist")

Project list:
[paste project names and rough notes]

Style:
- plain English
- no fluff
- outcome-focused
```

## Prompt 4: Story Arc (Career Pivot)

```text
Help me write a career pivot story that feels authentic.

Output format:
1) 1-line hook
2) 5-bullet timeline (year + milestone)
3) 120-word narrative paragraph
4) 1-line "why now" statement

My timeline:
[1980 ...]
[1989 ...]
[2001 ...]
[2025 ...]

Constraints:
- Keep facts concrete.
- No exaggerated claims.
- Emphasize continuity, not reinvention theatre.
```

## Prompt 5: Collaboration Invite

```text
Write 8 collaboration invite lines I can use at the bottom of my profile.

Audience:
- [e.g., neurodivergent creators]
- [e.g., ministry leaders]
- [e.g., experienced builders returning to tech]

Tone:
- humble, clear, welcoming
- not salesy

Each line:
- max 20 words
- includes one concrete collaboration angle
```
